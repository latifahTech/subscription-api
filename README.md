## <a name="introduction">🤖 Introduction</a>

A production-ready **Subscription Management System API** designed to handle authentication, database modeling, and real-world business logic securely and efficiently. The API is structured following clean architecture principles to ensure scalability and seamless integration with any frontend client.

---

## <a name="tech-stack">⚙️ Tech Stack</a>

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database & ODM:** MongoDB & Mongoose
- **Security & Protection:** Arcjet
- **Automation & Workflows:** Upstash QStash

---

## <a name="features">🔋 Features</a>

👉 **Advanced Rate Limiting & Bot Protection:** Secured using Arcjet to prevent automated attacks and control traffic.  
👉 **Database Modeling:** Well-structured schemas and relations using MongoDB & Mongoose.  
👉 **JWT Authentication:** Secure user registration, login, and token-based route protection.  
👉 **Subscription Management:** Full CRUD operations for tracking user subscriptions.  
👉 **Global Error Handling:** Clean validation middleware and centralized error responses.  
👉 **Automated Email Reminders:** Smart email workflow automation using Upstash.
